# A bayesian naive bayes classifier

Naive bayes classifiers are commonly-used tools in subject classification of text as well as in target identification in chemoinformatics. See [Jake VanderPlas' python handbook](https://jakevdp.github.io/PythonDataScienceHandbook/05.05-naive-bayes.html) for a practical example if you don't know what that means.
